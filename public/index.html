API 文档

欢迎访问 API 文档页面。以下是可用接口的详细信息和使用方法。

1. 获取分类列表

接口: https://imgapi.taoisms.org.cn/api/image?action=list&key=你的API密钥

此接口返回所有可用分类的列表。

2. 根据分类获取图片

接口: https://imgapi.taoisms.org.cn/api/image?action=get_images&cid=分类ID&key=你的API密钥

将 分类ID 替换为你想要获取图片的分类 ID，将 你的API密钥 替换为有效的 API 密钥。

此接口返回指定分类中的一张随机图片。

示例请求


curl -X GET "https://imgapi.taoisms.org.cn/api/image?action=list&key=你的API密钥"

curl -X GET "https://imgapi.taoisms.org.cn/api/image?action=get_images&cid=6&key=你的API密钥"

如何获取 API 密钥

请联系管理员以获取 API 密钥。
